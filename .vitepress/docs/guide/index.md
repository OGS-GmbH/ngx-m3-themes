# Getting started

## Installation

- Node.js version 18 or higher.
- A package manager: e.g. npm, pnpm, ...

::: code-group

```sh [npm]
$ npm install @ogs-gmbh/ngx-m3-themes
```

```sh [pnpm]
$ pnpm add @ogs-gmbh/ngx-m3-themes
```

```sh [yarn]
$ yarn add @ogs-gmbh/ngx-m3-themes
```

```sh [bun]
$ bun add @ogs-gmbh/ngx-utils
```

:::
